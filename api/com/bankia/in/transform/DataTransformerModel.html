com.modelicious.in.transform

DataTransformerModel

abstract class DataTransformerModel[T <: TransformerState] extends AnyRef

Base class for all transformer model objects. It holds a var state of type T <: TransformerState which has the needed configuration for the whole transformation process, i.e. the transform and the model generated by the com.modelicious.in.transform.DataTransformer.fit method.
