com.modelicious.in.transform

DataTransformer

abstract class DataTransformer[T <: TransformerState] extends AnyRef

Base class for all transformer objects. It holds a var state of type T <: TransformerState which has the needed configuration for the whole transformation process, i.e. the transform and the model generated by the com.modelicious.in.transform.DataTransformer.fit method.
