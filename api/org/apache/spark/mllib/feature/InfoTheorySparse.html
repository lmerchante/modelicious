InfoTheorySparse

class InfoTheorySparse extends InfoTheory with Serializable

Class that computes histograms for the Info-Theory primitives and starts the selection process (sparse and high-dimensional version).

The constructor method caches a single attribute (usually the class) to be re-used in the next iterations as conditional variable. This also computes and caches the relevance values, and the marginal and joint proportions derived from this operation.
